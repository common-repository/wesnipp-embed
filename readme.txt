Embed a WeSnipp Snippet easily on your WordPress website

== Description ==

[WeSnipp](http://wesnipp.com/ "WeSNipp") Embed will allow you to easily embed snippet into your WordPress post.

This plugin adds a buton to the Editor and a quicktag button to the HTML Editor.

A shortcode is genarated from the form on the Editor and it will make a shortcode with empty parameters from the quicktag button.

== Installation ==

1. Download the latest zip file and extract the directory it contains.
2. Upload the directory to your /wp-content/plugins/ directory.
3. Activate the plugin on the Plugins menu in the WordPress Dashboard.

== Frequently Asked Questions ==

= What is WeSnipp? = 

[WeSnipp](http://wesnipp.com/ "WeSnipp") is based on exchange and collaboration between developers..

= What is a Snippet? = 

A snippet is short reusable code.

= Do I need a WeSnipp account to use this plugin? =

No, if a Snippet is public you can still embed it on your WordPress site.
